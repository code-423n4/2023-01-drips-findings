1. Adding return variable expends more gas.
   - Example:

    ```
    function driverAddress(uint32 driverId) public view returns (address driverAddr) {
        return _dripsHubStorage().driverAddresses[driverId];
    }
    ```
    - Suggestion: Just remove the return variable.
    ```
    function driverAddress(uint32 driverId) public view returns (address) {
        return _dripsHubStorage().driverAddresses[driverId];
    }
    ```
